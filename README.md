## Projects

This repository hosts documentation and source code for several archival tools developed by our team:

### [Instagram Profile Archiver](./insta-archiver)
A Python-based tool for archiving Instagram profiles, generating high-quality PDF reports with posts, captions, and metadata.

### [Linux Floppy Bridger](./linux-floppy-bridger)
A robust bash script for safely mounting and bridging legacy floppy disks on modern Linux systems for data preservation.

### [News Scrape Archiver](./news-scrape-archiver)
A tool for scraping and archiving articles from Dartmouth News, featuring date filtering and PDF generation.

### [YouTube Archiver](./youtube-archiver)
A utility to capture and organize metadata, comments, and thumbnails from YouTube videos and channels.

### [DOCX to TXT Converter (cl-project-1)](./cl-project-1)
A utility script for converting Microsoft Word documents (.docx) to plain text format.

### [Sandbox (sb-project-1)](./sb-project-1)
A development sandbox for testing and experimental features.
